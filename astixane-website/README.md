# Astixane Technologies Website

## Overview
Astixane Technologies is a modern website designed to showcase innovative IT solutions for businesses. The website features a clean and responsive design, providing users with easy navigation and access to information about services offered.

## Project Structure
The project consists of the following files and directories:

```
astixane-website
├── index.html          # Main HTML document for the website
├── css
│   └── style.css      # Styles for the website
├── js
│   └── script.js      # JavaScript functionality for interactive elements
├── assets
│   ├── fonts          # Directory for custom font files
│   └── svg            # Directory for SVG files (icons and graphics)
├── .gitignore         # Specifies files to be ignored by Git
└── README.md          # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd astixane-website
   ```

2. **Open the Project**
   Open `index.html` in your preferred web browser to view the website.

3. **Modify Styles**
   Edit `css/style.css` to customize the appearance of the website.

4. **Add JavaScript Functionality**
   Update `js/script.js` to implement any interactive features or animations.

5. **Fonts and SVGs**
   Place any custom font files in the `assets/fonts` directory and SVG files in the `assets/svg` directory.
